# Social-Network-Analysis-Practical

This repository contains practical implementations and analysis of various Social Network Analysis (SNA) algorithms using Python and Jupyter Notebooks. It covers both theoretical concepts and hands-on coding for better understanding of graph-based techniques.

---

## Structure

```
Social-Network-Analysis-Practical/
│
├── HITS-algorithm.ipynb
├── NETWORK-MEASURES.ipynb
├── network-measures-2.ipynb
├── sim-rank-algorithm-implementation.ipynb
├── eigen-vector-centrality-and-page-rank-algorithm.ipynb
├── updated-page-rank-algorithm.ipynb
├── ego-facebook-NETWORK-dataset-analysis-*.ipynb
└── README.md
```

---

## Topics 

### 1. Network Measures

* Degree Centrality
* Closeness Centrality
* Betweenness Centrality

### 2. PageRank Algorithm

* Basic PageRank implementation
* Improved PageRank techniques

### 3. HITS Algorithm

* Authority and Hub score computation

### 4. Eigenvector Centrality

* Node importance based on connections

### 5. SimRank Algorithm

* Similarity between nodes in a graph

### 6. Dataset Analysis

* Facebook Ego Network dataset analysis

## How to Run

1. Clone the repository:

```
git clone <your-repo-link>
cd Social-Network-Analysis-Practical
```

2. Launch Jupyter Notebook:

```
jupyter notebook
```

3. Open any `.ipynb` file and run all cells.
